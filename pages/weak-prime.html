Step-by-Step Breakdown

1

DH Exchange with Selected Prime

Alice and Bob perform a standard DH exchange using the selected prime size. An attacker observes the public values p, g, and Y_A.

2

Brute Force Begins

The attacker iterates x = 1, 2, 3, … and checks if g^x mod p equals Y_A. For small primes the key space is tiny — the correct x is found almost immediately.

3

Private Key Recovered (or Not)

For small primes: the attacker finds X_A and can compute the shared secret K = Y_B^X_A mod p — full session compromise. For 2048-bit: the key space is 2^2048 ≈ 10^617 — brute force is physically impossible.
